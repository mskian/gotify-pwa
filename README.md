# Gotify PWA

Simple Web App with PWA Feature for Send Push Notifications to the Gotify Server.

> Demo - <https://push.santhoshveer.com>

## Features

- LocalStorage for Store the API on Browser
- Axios for Accessing the API
- Progressive Web App
